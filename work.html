Chat App

This React Native mobile app enables users to customize their chat experience by choosing a name and background color, then facilitates real-time messaging, image sharing, and location sharing. Integrated with Firebase/Firestore for data storage, this app provides users with both offline and online functionality.

This app was built in July, 2024.

Meet App

This React PWA is an event management tool, built using TDD. It integrates Google Calendar API with serverless functions for real-time event updates, and authenticates users via Google OAuth 2.0. Users can search for events, customize search criteria, visualize event data with dynamic charts, and access key features offline.

This app was built in June, 2024.

MyMovies App

This is a full-stack MERN app where users can securely sign up or log in to manage and interact with movie data. Authenticated users can view, edit, and delete movies from a favorites list, as well as manage profile details. The frontend is built with React, uses Fuse for dynamic search functionality, and integrates a custom-built API.

This app was built in April, 2024.

MyMovies-Angular App

In the interest of exploring Angular, I rebuilt the frontend of the MyMovies app, and integrated the same custom-built RESTful API. This new interface uses Angular Material, and still lets users perform CRUD operations. This app is built as a responsive SPA, allowing users to scroll sideways similar to modern movie apps.

This app was built in August, 2024.

Lists App

This React Native and ExpoGo-based mobile app manages lists. Users can create, update, and view their lists. It supports offline functionality (AsyncStorage) for a better user experience. Firebase Firestore is used for backend services, providing real-time data synchronization and offline data persistence.

This app was built in July, 2024.

Planting by Zones App

This JavaScript app simplifies gardening. Users input a zipcode and a third-party API fetches and displays detailed information about the user's hardiness zone. Users can also access links to additional information about their plants. All is built with user-friendly UI via Bootstrap.

This app was built in March, 2024.
